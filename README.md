# FRC 2026 Scouting App

This is the official scouting app for the 2026 season.  
It is deployed using **GitHub Pages** and works fully **offline** as a PWA.

## Features
- Offline-capable Progressive Web App (PWA)
- QR code export for match data
- Email/SMS/Clipboard sharing
- Fully mobile-friendly UI
- Works on Android and iOS
- No server required at competition

## Installation (Scouts)
1. Open the app URL in Chrome/Safari.
2. Tap **Install App** or **Add to Home Screen**.
3. Use the app offline all day.

## Development
All files are static and live in the repo root:
- `index.html`
- `app.js`
- `styles.css`
- `manifest.json`
- `service-worker.js`

## Deployment
GitHub Pages automatically deploys from the `main` branch.
